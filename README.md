# SpLibrary

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 17.2.0.

## Sp-Lib

This project is not a web application but essentially contains the npm library "sp-lib". It has two main sub-projects:

- Sp Library: This is the Angular library project, which contains reusable standalone components that can be integrated into other Angular projects.
- Sp Showcase: This is the Angular application that demonstrates the usage of the sp-lib library. It acts as a showcase project where you can test and visualize how the components from the sp-lib library work in a real Angular application.

## About

For more information about the sp-lib library, navigate to the library project folder located in /projects/sp-lib and check the README file inside that folder for details on how to use and integrate the library.
